Internationalization ‚Äã

Element Plus components are using English by default, if you wish you use other languages, you can get you answer by keep reading.

Global configuration ‚Äã

Element Plus provides global configurations

main.ts
ts
import ElementPlus from 'element-plus'
import zhCn from 'element-plus/es/locale/lang/zh-cn'

app.use(ElementPlus, {
  locale: zhCn,
})

ConfigProvider ‚Äã

Element Plus also provides a Vue component ConfigProvider for globally configuring locale and other settings.

App.vue
vue
<template>
  <el-config-provider :locale="zhCn">
    <app />
  </el-config-provider>
</template>

<script setup lang="ts">
import { ElConfigProvider } from 'element-plus'
import zhCn from 'element-plus/es/locale/lang/zh-cn'
</script>

Date and time localization ‚Äã

We use Day.js library to manage date and time in components like DatePicker. It is important to set a proper locale in Day.js to make the internationalization fully works. You have to import Day.js's locale config separately.

ts
import 'dayjs/locale/zh-cn'

CDN Usage ‚Äã

If you are using Element Plus via CDN, then you need to do this, let's again take unpkg as an example

html
<script src="//unpkg.com/element-plus/dist/locale/zh-cn"></script>
<script>
  app.use(ElementPlus, {
    locale: ElementPlusLocaleZhCn,
  })
</script>

Full documentation refer to: ConfigProvider
