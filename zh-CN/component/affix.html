Affix 固钉 #

将页面元素固定在特定可视区域。

Basic Usage #

固钉默认固定在页面顶部。

Target Container #

通过设置 target 属性，让固钉始终保持在容器内， 超过范围则隐藏。

Fixed Position #

Affix 组件提供 2 个固定的位置参数 top 和 bottom。
